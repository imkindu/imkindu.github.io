mysql_secure_installation

发表于 2017-09-22 | 分类于 mysql

　　mysql安装完成后，系统自带了几个用户和匿名用户可以登录，但是为了安全考虑，我们需要执行一遍mysql_secure_installation安全向导初始化一下我们的数据库。

　　刚安装完mysql，我们直接执行mysql命令就可以登录，默认使用的是root@localhost 空密码登录的，安全的第一件事就是给root这个超级霸主设置密码，使用mysql_secure_installation可以初始化让我们设置密码，还能限制root从远程登录。

文件位置

　　初始化脚本mysql_secure_installation放在哪里呢？默认在安装目录下的/bin目录下。
